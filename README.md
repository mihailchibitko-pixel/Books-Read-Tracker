# Трекер прочитанных книг

Консольное приложение на Python для персонального учёта прочитанных книг с модульной архитектурой и автоматической проверкой дубликатов.

## Структура модулей
- **main.py** — точка входа в приложение, консольное интерактивное меню и валидация пользовательского ввода
- **storage.py** — модуль сохранения и загрузки данных в формате JSON. Реализует быструю фильтрацию дубликатов с использованием хэш-множества 
- **stats.py** — аналитический модуль для расчёта средней оценки книг и вывода статистики популярности по авторам.
- **models.py** — описание базового класса `Book` с методами сериализации (`to_dict`) и десериализации (`from_dict`) объектов.

## Как запустить приложение
Для запуска программы выполните команду в терминале:
```bash
py main.py
```
